# 📊 Sales Data Analysis using Pandas & Matplotlib

## 📌 Project Overview

This project analyzes a company's sales and advertising data using **Python**, **Pandas**, and **Matplotlib**. It demonstrates how to load data, perform exploratory data analysis (EDA), generate business insights, and visualize trends using different types of charts.

This project is designed for beginners who want to practice data analysis before moving on to machine learning.

---

## 🚀 Features

* Load sales data from a CSV file
* Display and inspect the dataset
* Calculate total and average sales
* Find the highest and lowest sales days
* Calculate average advertisement spending
* Compute total advertisement spend per day
* Filter days with sales greater than a specified value
* Generate a correlation matrix
* Visualize data using:

  * Line Chart
  * Bar Chart
  * Scatter Plot
  * Pie Chart
* Print business insights based on the analysis

```bash

```

2. Place `sales_data.csv` in the project folder.

3. Open `sales_analysis.ipynb` in Jupyter Notebook.

4. Run the notebook cells.

## 📊 Dataset

The dataset contains the following columns:

| Column           | Description                     |
| ---------------- | ------------------------------- |
| Day              | Day number                      |

| Social_Media_Ads | Social media advertising budget |
| Newspaper_Ads    | Newspaper advertising budget    |
